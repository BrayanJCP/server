Desarrollar Front-End

Para iniciar por un buen camino para ser un desarrollador front-end es comenzar por lo basico (Como lo dice los profesores de Universidad).

En este caso es recomendable inicar con la maquetación de paginas Web, con ayuda de HTML y CSS (no son lenguajes de programacion). Esto no quiere decir que con esto seamos desarrolladores, el objetivo es conocer las etiquetas correspondientes para manejar la informacion que algun servicio web nos envia, esto, en mi opinion, es importante debido a que con esto sabemos lo que el programador desea mostrar, ya sea un parrafo o algun texto que este dentro de una caja y esta caja tenga un borde y un color. W3schools es una pagina excelente para conocer todo los conceptos basicos sobre la maquetacion con HTML y CSS.

Ahora si viene la magia... Despues de tener claro los conceptos de maquetación de paginas web debes conocer un lenguaje amigable, 'Javascript', con esto puedes ser un Dios literal, puedes hacer que tu texto cambien, darle animación(con css tambien lo puedes lograr), puedes capturar la informacion que viene desde algun formulario y hacer el tratamiento que quieras; con w3schools puesde ver tambien como funciona todo este mundo, es enserio interesante.
