Desarrollar Front-End (Web)

Para adquirir los conocimientos básicos de un desarrollador front-end se debe tener claro los conceptos de html, css y javascript:

  • HTML: HyperText Markup Language (lenguaje de marcas de hipertexto). Básicamente es un documento web (.html), el cual contiene una serie de etiquetas que reconocen los navegadores(Chrome, Firefox, Internet Explorer, entre otros) para organizar la información(Por lo general son textos e imagenes).

  • CSS: es un mecanismo simple para añadir estilos a un documento web(HTML o XML).

  • Javascript: Es un lenguaje de programación que permite hacer el procesamiento de la información.
