Desarrollar Front-End

Para iniciar por un buen camino para ser un buen desarrollador front-end es comenzar por lo basico (Como lo dice los procesores de Universidad).

En este caso es recomendable inicar con la maquetacion de paginas Web, con ayuda de HTML y CSS (no son lenguajes de programacion). Esto no quiere decir que con esto seamos desarrolladores, el objetivo conocer la etiquetas correspondientes para manejar la informacion que algun servicio web, esto, en mi opinion, es importante debido a que con esto sabemos lo que el programador desea mostrar ya sea un parrafo, que este detro de una caja y esta caja tenga un borde y un color. W3schools es una pagina excelente para conocer todo los conceptos basicos sobre la maquetacion.
